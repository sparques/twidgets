# twidgets

The twidgets package provides additional widgets for tview:

  - Label: simple way to add text to an application without the overhead of a TextView.
  - Map: Show spatially related dated. 
  - progressbar: a progress bar with horizontal and vertical orientation and optional end-caps.
  - mdview: a textview-like widget that renders markdown to tcell styles.
  - Scrollbar: general purpose horizontal or vertical scroll bar; can be used with TextView, TextArea, Tables, etc.
